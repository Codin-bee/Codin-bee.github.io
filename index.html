Projects

SNN4J

Simple Neuron Networks for Java (SNN4J) is my machine learning library developed from scratch. It allows users to create, train, and utilize multi-layer perceptrons (fully connected feedforward networks) with just a few lines of code. The primary goal is to serve as a learning resource for those new to AI, so it isn’t highly optimized; instead, the code is designed to reflect how this topic is typically explained in textbooks, helping users grasp the concepts more easily.

Check out SNN4J

Digit Recognizer

Simple project for digit recognition showcasing the ability of my SNN4J library. Using a small MLP from the library, only a small part of the MNIST database, and 100 or so iterations, it managed to get 75% accuracy on a testing dataset with 20 thousand examples.

Check out Digit Recognition

Note Transformer

A custom Transformer architecture designed to generate simple piano music, built from scratch to reduce the number of parameters while maintaining high performance. It features custom embeddings, a specialized layer for handling timing, and a custom unembedding process. Future updates will incorporate multiple datasets to improve generalization and music quality.

Check out Note Transformer
